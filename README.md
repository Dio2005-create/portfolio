# Portfolio Professionnel React

Un portfolio moderne prêt à l'emploi pour un développeur web et data engineer, construit avec React, Vite et Tailwind CSS.

## Installation

1. Installer les dépendances :
   ```bash
   npm install
   ```
2. Lancer le serveur de développement :
   ```bash
   npm run dev
   ```

## Construction

Pour créer la version de production :
```bash
npm run build
```

## Contenu

- Page d'accueil avec titre, présentation et boutons d'action
- Section "À propos" avec compétences et profil
- Section "Projets" avec cartes descriptives
- Section "Compétences" avec barres de progression
- Section "Contact" avec formulaire et liens GitHub / LinkedIn
- Footer professionnel

## Personnalisation

Modifiez `src/App.jsx` pour remplacer les informations fictives par vos données personnelles.
